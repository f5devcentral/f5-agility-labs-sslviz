Lab 1: Inbound Gateway Topology
================================================================================

In this module, you will deploy SSL Orchestrator as a reverse proxy in **L3 Inbound Gateway Mode** to perform TLS inspection for multiple backend applications. You will also leverage the **on-box F5 Advanced WAF** module as an inspection service to protect backend applications from common attacks.
