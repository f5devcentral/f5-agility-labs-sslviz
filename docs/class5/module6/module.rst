Lab 4 - Automate a Deployment via API
================================================================================

In this module, you will use the BIG-IP Next Central Manager API to create an SSL Orchestrator TAP inspection service, a service chain containing the TAP inspection service, and a simple traffic policy. The SSL Orchestrator policy will then be applied to an inbound application.
