Lab 2 - Inbound Application Mode Deployment
================================================================================

In this module, you will create an SSL Orchestrator inbound application mode deployment with an inline L3 inspection service, a service chain, and a simple traffic policy. The SSL Orchestrator traffic policy will then be applied to an inbound application.
