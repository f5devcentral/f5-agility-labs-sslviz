Lab 3 - Inbound Gateway Mode Deployment
================================================================================


In this module, you will create an SSL Orchestrator ICAP inspection service, a service chain containing multiple inspection services (Inline L3, ICAP), and a simple traffic policy. The SSL Orchestrator policy will then be applied to an **Inbound Gateway Mode** application.
