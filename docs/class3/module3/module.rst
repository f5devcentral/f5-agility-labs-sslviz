Manual SSL Orchestrator Topology Configuration
================================================================================

Now that the network infrastructure and VM instances have been deployed (using Terraform in the previous exercise), you can deploy an SSL Orchestrator **Topology**.

In this module, you will manually create and test an **Inbound Layer 3 Topology** with the following:

- SSL configuration
- Inline Layer 3 Services
- Service Chains
- Security Policy
- Interception Rule
- Egress SNAT
- Log Settings
- Application Pool
