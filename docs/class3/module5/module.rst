Nuking the Configuration
================================================================================

In this module, you will:

- Run an Ansible playbook to delete the SSL Orchestrator configuration.
- Run an Ansible playbook to revoke the BIG-IP VE license.
- Use the Terraform **destroy** command to completely remove the AWS deployment.
